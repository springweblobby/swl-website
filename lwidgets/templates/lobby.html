Spring is a free RTS engine developed for Windows, Linux and Mac OS X. Using Spring Web Lobby you can connect to the Spring multiplayer battle server.







Help

Windows

Cannot start command: Downloading Engine 96.0: assign: The parameter is incorrect

Are you on windows XP? Unfortunately the lobby does not work with Windows XP. Seek alternative lobby clients.

Linux

In order to play on linux you need to have SDL installed. Most probably you already have it, but on rare occasions it may not be there. In Ubuntu that is solved by installing the package called libsdl1.2debian, for other distros the name of the package will be different.
